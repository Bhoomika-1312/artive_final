# 🎨 Artive – Creative Art Platform

Artive is a modern web platform designed to showcase artworks, events, and creative talent. It provides users with an interactive interface to explore art, add their own creations, and engage with the community.

---

## 🚀 Features

* 🖼️ Browse and explore artworks
* ➕ Add new artwork
* 📅 View upcoming events
* ℹ️ Learn about the platform (About page)
* 👥 Meet the team
* 📱 Responsive UI using Tailwind CSS

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Build Tools:** PostCSS
* **Package Manager:** npm

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/artive.git
```

2. Navigate to the project:

```bash
cd artive
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm start
```
